Detailed Description

A basic class that provides a timespan.

A timespan defines an abstract amount of time that consits of a unit, e.g. seconds or minutes, and a count. The result could be for example 42 Seconds.

For now, the timespan is a very simple object, but in future versions it will be extended (and propably moved outside of this project) to reach it's full capability.
