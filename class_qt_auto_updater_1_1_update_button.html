Detailed Description

A simple button for update checks.

The UpdateButton is a ready-made "button" to check for updates. It is connected to an updater and thus will automatically start checking for updates. It will also display a progress and optionally allow canceling and installation of updates.

The button will represent the updaters state and automatically enable/disable itself to avoid start beeing called twice. In addition to that, the whole button widget will be disabled if the updater gets deleted while the button is still existant.

◆ ModeFlag

enum QtAutoUpdater::UpdateButton::ModeFlag
strong

Defines various flags to specify how the button should behave.

Enumerator
ShowResult 

Show a result (e.g. "new updates") after a check completed next to the button.

ShowStatus 

Show the check status as provided via Updater::progressChanged instead of a simple "checking for updates".

AllowInstall 

Allow the user to launch an installer via the button if updates are available.

AllowCancel 

Allow the user to cancel the update check.

None 

Value to enable none of these features.

AllowAndShowAll 

Enables all features at once.
